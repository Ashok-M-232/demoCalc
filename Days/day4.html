A computer network is a collection of devices connected to share resources, communicate, and exchange data securely.

SOC L1 Relevance

  • Monitor network traffic for malware, unauthorized access, and anomalies.
  • Understand network layout to assist incident response and threat hunting.

Components

  • Hardware Networking: Routers, switches, hubs, access points, cabling ‚Äî used for traffic monitoring, segmentation, device identification.
  • Software Networking: Protocols, network OS, and management tools ‚Äî monitor logs, protocol usage, anomalies.

5Ô∏è‚É£ IP Addresses & MAC Address

  • IPv4: 32-bit, limited (e.g., 192.168.1.10)
  • IPv6: 128-bit, vast (e.g., 2001:db8:85a3::8a2e)
  • Public IP: Monitor external attacks, reconnaissance, exfiltration
  • Private IP: Monitor lateral movement, unauthorized internal traffic
  • MAC Address: Monitor for spoofing, rogue devices

Tools: SIEM, NAC, Wireshark, IPAM

7Ô∏è‚É£ DNS (Domain Name System)

Converts domain names to IPs. SOC monitors for DNS-based attacks.

  • DNS spoofing / cache poisoning
  • DNS tunneling (malware exfiltration)
  • Malicious domains (DGA detection)

L1 Actions: Monitor query patterns, block malicious domains, escalate anomalies.
